# AgroGeo Module – Scope

## Что делаем
- Карта с объектами (бурт, фермер, пункт приёмки)
- Маршрут A → B
- Расчёт расстояния (км)
- Расчёт стоимости (по формуле)
- Интеграция с АгроCRM24 (MVP)
- Демо-режим

## Что НЕ делаем
- Оптимизация маршрутов
- Расчёт времени маршрута
- Мобильные приложения
- Использование зарубежных внешних API

## Ограничения
- Сервер и данные — РФ
- Используются open-source решения
